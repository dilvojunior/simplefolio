Projects

Find Wisdom

Find Wisdom is my first independent project, created entirely from scratch and inspired by my passion for books and the pursuit of wisdom. The concept is to develop an online library where users can search for books or authors and explore a wealth of knowledge. To build this project, I utilized React for the frontend and integrated the Google Books API as the source for book data. This allowed me to work with real-world APIs and improve my skills in building interactive applications. This project represents not only my technical abilities but also my ability to bring personal interests into meaningful development work. Although it’s still a work in progress, I’m continuously refining it and expanding its features to enhance the user experience.
